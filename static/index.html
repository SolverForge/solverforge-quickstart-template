REST API Guide

Integration via cURL

1. Download demo data

curl -X GET http://localhost:8080/demo-data/SMALL -o sample.json

2. Start solving

Returns a jobId for subsequent requests.

curl -X POST -H 'Content-Type:application/json' http://localhost:8080/schedules -d @sample.json

3. Check status

curl -X GET http://localhost:8080/schedules/{jobId}/status

4. Get solution

curl -X GET http://localhost:8080/schedules/{jobId}

5. Stop solving

curl -X DELETE http://localhost:8080/schedules/{jobId}
